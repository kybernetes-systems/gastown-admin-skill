# Minimal Skill

This skill demonstrates the smallest valid Agent Skill: frontmatter with `name` and `description`, plus one paragraph of instructional body text. No references, no scripts, no assets—just essential capability delivered to the agent with zero overhead.

**Produces:** Clear, actionable instructions for the agent to complete a task.

This is your stripped-down War Rig: every component serves a purpose. If you can say it in one paragraph, say it. If you cannot, move material to `references/` and link it from the body.
